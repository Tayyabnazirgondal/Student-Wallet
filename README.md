# 💰 Student Wallet

**Student Wallet** is a simple, private, and fully offline expense-tracker mobile app designed especially for students to manage their pocket money, track spending, and build healthy financial habits — no login, no data collection.

Developed by **Daynix Studio**.

---

## ✨ Features

- 📊 **Dashboard** — See total balance, income, and expenses at a glance
- ➕ **Add Transactions** — Record income and expenses by category
- 🎯 **Monthly Budget** — Set a budget and get warnings when you overspend
- 📈 **Statistics** — Visual spending breakdown with pie charts
- 🔍 **Search** — Quickly find any transaction by title or category
- 💵 **Multi-Currency** — Choose your currency (works worldwide)
- 👤 **Profile** — Set your name, date of birth, and profile picture
- 🌙 **Dark Mode** — Comfortable viewing, choice saved automatically
- 🔒 **100% Offline & Private** — No internet, no account, data stays on your device

---

## 🛠️ Built With

- **Flutter** — Cross-platform UI framework
- **Hive** — Fast, lightweight local database
- **Provider** — State management
- **fl_chart** — Charts and data visualization
- **shared_preferences** — Local settings storage
- **image_picker** — Profile photo selection
